COMO SOLICITAR TU PAGINA WEB

Dirigirse a la pagina Contacto y llenar los campos que se visualizan. En el campo de texto donde pondras la informacion debes indicarme brevemente tu consulta.

Una vez entremos en contacto me plantearas tu idea para tu pagina web de forma detallada para asi poder cotizar y empezar con el proyecto.

Luego de realizada la idea se estimara un plazo para realizar dicha pagina web poniendose de acuerdo con el comprador.

Ya finalizado el plazo de entrega y el proyecto se comunicara con el comprador para realizarse la entrega del trabajo.

Por ultimo, el pago final se podra hacer por transferencia bancaria o mediante PayPal.
